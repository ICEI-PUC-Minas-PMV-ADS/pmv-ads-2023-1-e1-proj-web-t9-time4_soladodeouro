# Template padrão do site

O template é composto pelos seguintes layouts:
- **Tela principal**
  - **Lista de Calçados**
    - **Informação do Calçado**
    - **Compra** 
- **Fórum**
- **Menu de navegação**
  - **Sobre nós**
  - **Fale conosco**
  - **Login**
  - **Entrar/Cadastrar**

Tela que abrange todas as visualizações iniciais do site.
